# 15x15 mcts8 v1 evaluation summary

Checkpoint:
- checkpoints/15x15_mcts8_v1.pt

Evaluation settings:
- board_size: 15
- win_length: 5
- eval mcts_sims: 4
- use_safety: False

Results:
- v1 vs random: 1W / 0D / 3L, avg_moves=122.75
- v1 vs smoke checkpoint: 2W / 0D / 2L, avg_moves=84.00
- v1 vs greedy_win_block: 0W / 0D / 4L, avg_moves=21.50

Interpretation:
- The 15x15 pipeline works, but this first baseline is not strong.
- v1 is not clearly better than the smoke checkpoint.
- v1 loses quickly to a simple greedy win/block baseline, so it has not learned stable tactical blocking.
- This is expected because v1 was trained with limited self-play data and low MCTS simulations.
- Next step should be stronger training, not more hard-coded C safety rules.
