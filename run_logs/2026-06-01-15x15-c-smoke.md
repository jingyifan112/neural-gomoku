Goal:
- Verify that the Python 15x15 checkpoint can be exported to C-readable weights.
- Verify that the C inference build can be temporarily switched from 9x9 to 15x15.
- Verify that pbrain-neural-gomoku accepts START 15.

Checkpoint:
- checkpoints/15x15_smoke.pt

Exported weights:
- c_inference/weights/15x15_smoke_weights.bin
- c_inference/weights/15x15_smoke_weights_manifest.txt

Expected manifest:
- board_size: 15
- input_shape: [3, 15, 15]
- total_floats: 413415
- total_bytes: 1653660

C changes:
- GOMOKU_BOARD_SIZE changed from 9 to 15.
- GOMOKU_BOARD_CELLS changed from 81 to 225.
- GOMOKU_WEIGHT_FLOATS changed from 315927 to 413415.
- pbrain default weights changed to weights/15x15_smoke_weights.bin.

Conclusion:
- This is only a 15x15 smoke build.
- The model is not expected to be strong yet.
- No new hard-coded tactical safety rules were added.
